## 🎯 목적

- C 언어를 단순 문법이 아닌 컴퓨터 동작 관점에서 이해
- 강의를 따라 구현하며 개념을 체화
- 포인터, 메모리, 구조체 등 핵심 개념을 실습 기반으로 정리
- 그래픽스 / 시스템 프로그래밍으로 확장 가능한 기반 구축
